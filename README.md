# eightChar-web
国学命理app-h5

本项目为基于Vue3开发的h5八字排盘app，可根据已知时间、四柱进行排盘！
排盘界面分命主信息、基本命盘、顺物。
排盘数据包含四柱、十神、藏干、纳音、星运、空亡、神煞、天罡称骨、五行数据等信息。

技术栈： vue3 + typeScript + vite

组件库： vant
